## Amigo Secreto

Projeto desenvolvido em **`JavaScript, HTML e CSS`** para realizar o sorteio de **Amigo Secreto**, permitindo adicionar participantes, validar entradas e gerar automaticamente os pares do sorteio.

### ⚙️ Funcionalidades
- Adição de nomes de participantes
- Validação de campo vazio
- Impede nomes duplicados
- Sorteio automático sem auto-sorteio
- Embaralhamento aleatório dos participantes
- Opção de reiniciar o sorteio
